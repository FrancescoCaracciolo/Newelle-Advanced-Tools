# Advanced Tools Extension for Newelle

This extension enhances Newelle with powerful file system operations, search capabilities, and utility tools.

## Features

### File Operations
- **Read File**: Read file contents with optional line range and line numbering
- **Write File**: Create or overwrite files, with support for line-specific editing
- **Search and Replace**: Targeted text replacement within files
- **Delete File**: Safely remove files (empty directories only)
- **Copy File**: Duplicate files from source to destination
- **Rename/Move File**: Rename or move files
- **Create Directory**: Create new directories (including parent directories)

### Search Capabilities
- **Semantic Search**: Intelligent search across documents, files, folders, and websites
- **Grep**: Fast pattern matching in files with regex support
- **Glob File Search**: Find files matching specific name patterns

### System Utilities
- **List Directory**: View directory contents
- **Change Directory**: Navigate the file system with UI integration
- **Download File**: Download files from URLs
- **Analyze Image**: Extract information from images using vision-capable LLMs

### Configuration Options
- **Max Output Length**: Control response length (1000-100000 characters)
- **Secondary LLM for Image Analysis**: Toggle between primary/secondary LLMs
- **Image Analysis Prompt**: Customize the prompt used for image analysis
